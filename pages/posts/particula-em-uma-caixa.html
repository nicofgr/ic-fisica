POÇO QUADRADO INFINITO


O problema do poço quadrado infinito, também conhecido como partícula em uma caixa, é um problema de fácil solução usado para demonstrar o comportamento de corpos descritos pela equação de onda. $$\color{blue}{-\frac{\hbar^2}{2m} \psi''(x) + V(x)\psi(x)=E\psi(x)}$$ O poço quadrado infinito consiste em um corpo constrito em uma região de potencial nulo entre duas barreiras de potencial infinito.

IMAGEM

Como a região entre as barreiras é nulo, ou seja, $V(x) = 0$, a equação de onda que descreve essa região é: $${-\frac{\hbar^2}{2m} \psi''(x) + \cancelto{0}{V(x)\psi(x)}=E\psi(x)}$$ $$-\frac{\hbar^2}{2m} \psi''(x) =E\psi(x)$$ $$\psi''(x) =-\frac{2mE}{\hbar^2}\psi(x)$$ $$\psi''(x) =-k^2\psi(x)$$ $$\psi''(x) + k^2\psi(x) = 0$$ A solução dessa equação diferencial linear de segunda ordem é do formato: $$\psi(x) = A \sin{(kx)} + B \cos{(kx)}$$ As condições de contorno: $$\begin{align} \psi(0) = 0 && \psi(L)=(0) \end{align}$$

Detalhes da solução

$$\begin{gather} \psi(0) = 0 && \psi(L)=(0) \\ \psi(0) = A \sin{(k0)} + B \cos{(k0)} = 0 && \psi(L) = A \sin{(kL)} + B \cos{(kL)} = 0\\ \psi(0) = \cancelto{0}{A \sin{(k0)}} + B \cancelto{1}{\cos{(k0)}} = 0 && \vdots\\ \psi(0) = B = 0 && \vdots \\ \boxed{B = 0} && \psi(L) = A \sin{(kL)} + \cancelto{0}{0 \cos{(kL)}} = 0\\ && \psi(L) = A \sin{(kL)} = 0\\ \end{gather}$$ $A \ne 0$ pois caso contrário $\psi(x) = 0$ e não haveria partícula, logo: $$\begin{gather} \sin(kL) = 0 \\ kL = n\pi \\ k = n \frac{\pi}{L} \\ \end{gather}$$ Normalizando a função de 0 a L: $$\begin{gather} \langle \psi | \psi\rangle = \int_{-\infty}^{+\infty} \psi^* \psi \ dx = 1\\ \int_{0}^{L} A^2 \sin^2\left(\frac{n\pi x}{L}\right) \ dx = 1\\ A = \sqrt{\frac{2}{L}} \end{gather}$$



Aplicando as condições de contorno conseguimos a solução: $$\begin{align} \psi_n(x) = \sqrt{\frac{2}{L}} \sin\frac{ \color{blue}{n} \color{black}{\pi x}}{L} && n = 1,2,3... \end{align}$$ No qual $n$ é o número quântico, e está relacionado à energia através da equação: $$E_n = n^2 \frac{\hbar^2\pi^2}{2mL^2} = n^2 E_1$$
